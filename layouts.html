Landmark Regions

The following HTML sectioning elements create ARIA landmark regions.

HTML Element Implied Landmark Role

footer if it is in the context of a body element contentinfo if it is in the context of a body element
header banner
main main
nav navigation
section if it has an accessible name provided by aria-labelledby or aria-label region
